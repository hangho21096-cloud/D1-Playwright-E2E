# Manual Test Cases

Thư mục này dành riêng cho việc lưu trữ các tài liệu kịch bản kiểm thử thủ công (Manual Test Cases).

Các Agent (như `test-impl`) có thể được cấu hình để sinh ra các file `.csv` hoặc `.md` chứa các bước test chi tiết (step-by-step) vào thư mục này để đội ngũ QA/QC thực hiện test bằng tay trên thiết bị thực (Mobile) hoặc Web.

Ví dụ:
- `login-manual-tests.md`
- `payment-flow-mobile.csv`
